About Link Pages

The info in the URL changes what's on the page (aka URL parameters).

For example:
id=50377515&name=recipes&icon=cookie-bite&theme=73d563&bg=f85519

id = raindrop link collection
name = page name
icon = Font Awesome Icon

*Colors must have the # removed
theme = main page color (must be hex code without the # - use my tool)
bg = background color (must be hex code without the # - use my tool)

Another color picker with more color options (just remove the #)
